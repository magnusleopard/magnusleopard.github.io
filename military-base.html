Halo Spartan Laser

Personal project

Video with breakdown below. This is a project I did during the 16 weeks in the VFX Professional Course in The Animation Workshop. I took the live action footage and added CG elements on top of it. All the work from beginning to end was done by me. The biggest focus point for me was asset creation, so I wanted to create a hero asset in front with multiple background elements. All of this in a sci-fi environment. So far it's the largest and most interesting project I've done and I learned a lot during this time.

Also big thanks to the teachers and fellow students!

    Software used:

  • Maya
  • Substance Painter
  • Nuke
  • Arnold
